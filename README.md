# 📖 Project Overview

This repository presents the complete engineering design of an **Off-Grid Solar Photovoltaic System** developed using **PVsyst 7.4**.

The project includes electrical sizing, battery bank design, PV array configuration, MPPT controller sizing, single-line diagram, techno-economic analysis, and complete engineering documentation following IEEE reporting standards.

---

# 🎯 Objectives

- Design a reliable Off-Grid Solar PV System
- Meet a daily load demand of **52.5 kWh/day**
- Achieve **1-Day Battery Autonomy**
- Perform detailed PV sizing
- Battery Bank Design
- MPPT Controller Selection
- Electrical Protection Design
- Techno-Economic Analysis
- Cash Flow Analysis
- IEEE Style Documentation
